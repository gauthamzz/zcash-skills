# Basic Research

Research enough to answer well. Do not over-collect junk.

## What this skill does

- gather a small set of relevant sources
- compare source quality and freshness
- extract the defensible claims
- pass clean findings to writing or explanation work

## Example requests

- "Research this claim before I answer."
- "Compare the sources and tell me what is solid."
- "Summarize the latest credible information on this topic."

## Working style

1. Restate the question.
2. Prefer primary sources first.
3. Compare agreement and disagreement.
4. Answer with attribution and uncertainty.

## Guardrails

- Do not bury source conflicts.
- Do not confuse recent with authoritative.
- Keep the source set tight.
