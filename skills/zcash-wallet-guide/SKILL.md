# Zcash Wallet Guide

Help the user do the wallet job safely and clearly.

## What this skill does

- help choose a wallet path
- explain address and receiver concepts
- guide receive and send workflows
- surface privacy and backup caveats

## Example requests

- "Which wallet should I use for Zcash?"
- "How do I receive ZEC without screwing up privacy?"
- "What do I need to back up?"

## Working style

1. Identify the user goal.
2. Give the shortest safe workflow.
3. Name the main privacy or safety caveat.
4. Route to node or RPC skills when the question stops being wallet UX.

## Guardrails

- Do not promise wallet feature support you have not verified.
- Do not recommend sloppy seed phrase handling.
- Do not blur the line between custody, compatibility, and privacy.
