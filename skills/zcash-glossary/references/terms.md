# Core Terms

## Shielded

A shielded transfer hides important transaction data with zero-knowledge proofs.

## Transparent

A transparent transfer exposes more transaction data publicly on-chain.

## Sapling

An older but still important shielded pool and protocol era that improved shielded usability substantially.

## Orchard

A newer shielded pool associated with more recent Halo-era upgrades.

## Unified address

A single address format that can package multiple receiver types for compatibility.

## Receiver

A concrete payment target inside a broader address container.

## Note

A shielded value object, roughly analogous to a spendable output in the private system.

## Nullifier

A value revealed when spending so the network can prevent double-spends without exposing the original note.
