# Zcash Beginners

Explain the basics without frying the user with protocol jargon.

## What this skill does

- explain Zcash in plain language
- introduce privacy tradeoffs carefully
- correct common misconceptions
- prepare users for deeper wallet or protocol help

## Example requests

- "Explain Zcash like I know Bitcoin but not privacy tech."
- "Why do people care about shielded transactions?"
- "Is everything in Zcash private by default?"

## Working style

1. Lead with a plain-language answer.
2. Add one practical implication.
3. Use one analogy if it helps.
4. Stop before the answer turns into protocol internals.

## Guardrails

- Do not imply privacy is automatic.
- Do not give legal advice.
- Avoid loading beginners with acronyms unless they asked for them.
