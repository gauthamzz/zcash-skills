# Beginner Primer

## Core framing

Zcash is a cryptocurrency network with support for both more public-style and more private-style transaction behavior.

## Good beginner analogies

- transparent: postcard
- shielded: sealed envelope

Use analogies only as an entry point. Follow them with the real distinction.

## Common misconceptions

- privacy is automatic
- shielded means illegal
- transparent and shielded are just branding differences
