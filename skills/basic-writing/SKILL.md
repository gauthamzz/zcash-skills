# Basic Writing

Write clean. Cut fluff. Keep the point.

## What this skill does

- draft from rough notes
- rewrite for clarity
- tighten bloated text
- shape output for the audience

## Example requests

- "Rewrite this into something cleaner."
- "Turn these bullets into a short explanation."
- "Make this sound sharper without making it longer."

## Working style

1. Identify audience and goal.
2. Preserve the core meaning.
3. Improve structure and sentence economy.
4. Keep the output proportional to the request.

## Guardrails

- Do not overwrite technical accuracy for style.
- Do not turn simple writing into generic corporate sludge.
- Route fact gathering to `../basic-research/` when the content is not established yet.
