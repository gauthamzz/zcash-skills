# Recovery

Use this file when the user is asking what to back up, how to recover, or how much risk a support step introduces.

## Recovery topics

- what backup material exists
- what must be preserved before changing devices
- what support steps are safe versus risky
- when the user should stop and verify before proceeding

## Practical Zcash-specific caution

- in `zcashd`, backup guidance historically depended on keys and newly generated addresses; wallet-specific guidance can differ, so do not generalize node-wallet backup instructions to every mobile wallet
- if the user is on a modern mobile wallet, answer in that wallet’s support terms first and only bring in `zcashd` backup behavior when they are clearly using node wallet features

## Default answer shape

1. identify the asset or secret being discussed
2. state why it matters
3. state the safe handling rule
4. state what not to do
