# Zodl

Use this file when the user asks specifically about Zodl or when it is the most relevant real-world Zcash wallet example.

## Current framing

- Zodl is the current brand for the flagship Zcash wallet previously known as Zashi.
- The rebrand does not imply a new wallet migration flow by itself.
- Zodl is relevant for normal-user wallet support, shielded ZEC usage, and product examples around swaps, CrossPay, and wallet UX.

## Support guidance

- if the user says "Zashi", recognize that the wallet is now branded as Zodl
- avoid implying users must create a new wallet just because of the name change
- when discussing cross-chain or swap features, state clearly that these are broader product flows and not the same thing as a pure shielded ZEC transfer
