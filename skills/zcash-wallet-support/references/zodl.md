# Zodl

Use this file when the user asks specifically about Zodl or when it is the most relevant real-world Zcash wallet example.

## Current framing

- Zodl is the current brand for the flagship Zcash mobile wallet previously known as Zashi.
- the Zodl team states that the rebrand is automatic and does not require users to create a new wallet, move funds, or change seed phrases
- Zodl publicly positions itself around shielded ZEC payments, CrossPay, swaps powered by NEAR Intents, Keystone hardware-wallet support, and Flexa-powered retail payments

## Support guidance

- if the user says "Zashi", recognize that the wallet is now branded as Zodl
- avoid implying users must create a new wallet just because of the name change
- if the user asks about swaps or CrossPay, explain that these are broader product flows layered on top of Zcash usage, not the same thing as a pure shielded ZEC transfer
- if the user asks about hardware-wallet support, Zodl’s public site specifically names Keystone support for shielded ZEC management
